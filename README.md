# Nvim Lua Config
## Dependencies
**NPM** is necesary for some servers
```bash
sudo pacman -S neovim
```
## Copy this config
```bash
git clone git@github.com:Killer2324/nvim_config.git
mv nvim_config $HOME/.config/
cd $HOME/.config
mv nvim_config nvim
cd nvim
nvim init.lua
```
> Note: Packer will be installed if you don't have it
